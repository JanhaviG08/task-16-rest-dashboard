# Task 16 – REST API Data Dashboard

## Overview
Frontend dashboard consuming a public REST API using Fetch API.

Displays product data with filtering, sorting, and pagination.

---

## API Used
https://dummyjson.com/products

---

## Features
- Fetch API integration
- Product listing
- Search filter
- Sort by price or rating
- Pagination
- Error handling
- Responsive grid layout

---

## Learning Outcomes
- REST API consumption
- Async JavaScript
- DOM rendering
- Pagination logic
- Filtering & sorting arrays
- Error handling
